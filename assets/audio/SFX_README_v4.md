# Audio Files

The game uses Phaser audio cues from `assets/audio/`.

Important files:

- `boss_bgm_protocol_for_nothing.mp3`: Boss battle BGM
- `The_Weight_of_Stone.mp3`: Main menu BGM
- `player_dash_clean.wav`: Dash feedback
- `player_blink_clean.wav`: Blink feedback
- `player_hurt_clean.wav`: Player damage feedback
- `tree_barrier_raise_clean.wav`: White tree growth
- `tree_barrier_break_clean.wav`: White tree damage/break
- `turret_deploy_clean.wav`: Mass-Energy Turret deployment
- `turret_shot_clean.wav`: Turret fire
- `drone_locking_beep_clean.wav`: Air raid targeting warning
- `drone_bomb_explosion_clean.wav`: Bomb impact
